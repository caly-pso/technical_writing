Layout

A simple one-page layout for the portfolio to ensure ease of use and a uniform experience.

Include Sections Such As

  • Navagation Bar
  • Projects Section
  • About Section
  • Resume Section
  • Footer/Contact Me Section

Design

  • Separate each section with a title
  • Inlcude a persisting 'back to top' button
  • Make the navagation bar easy to access while scrolling thorugh the page.
